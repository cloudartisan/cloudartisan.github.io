# Shrnk.ai

A modern URL shortening service with AI-powered features.

## Features

- Fast URL shortening
- Custom short URLs
- Analytics dashboard
- API for developers
- AI-powered link categorization

## Technology Stack

- Frontend: React
- Backend: Node.js
- Database: PostgreSQL
- Hosting: AWS
