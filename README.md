# cloudartisan.github.io

My blog - the site, the content and the code

## Installation

```
brew install hugo
```

## Development

To serve locally, including drafts:

```
hugo server -D
```

Visit http://localhost:1313/ to see the site.

## Static Site

### Build

```
hugo
```
